## Descripción del proyecto

Sistema de seguridad electrónica basado en la integración de dispositivos IoT sobre arquitecturas de red seguras, orientado a viviendas, empresas y conjuntos residenciales del municipio de Yopal, Casanare.

El proyecto no desarrolla software propio ni fabrica hardware. Integra tecnología comercialmente disponible — cámaras IP, sensores PIR, sensores de apertura y un grabador de red (NVR) — dentro de una red segmentada por VLAN con cifrado TLS 1.2 en todas las comunicaciones.

**Nivel TRL alcanzado:** TRL5 — validación en entorno real (centro de salud, Yopal)
**Metodología de desarrollo:** Scrum (5 sprints de 2 semanas)
**Marco formativo:** CDIO
